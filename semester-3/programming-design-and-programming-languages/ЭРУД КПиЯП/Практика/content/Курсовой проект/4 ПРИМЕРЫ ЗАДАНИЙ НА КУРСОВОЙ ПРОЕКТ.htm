4 ПРИМЕРЫ ЗАДАНИЙ НА КУРСОВОЙ ПРОЕКТ

 

Курсовой проект представляет собой задачу по проектированию прикладного программного обеспечения из следующих основных типов:

·       задачи, связанные с математическими расчетами;

·       задачи автоматизации действий пользователя;

·       задачи планирования;

·       задачи моделирования.

Деление задач достаточно условное, существует пересечение между перечисленными группами задач.

 

К задачам, связанным с математическими расчетами, можно отнести приложения, выполняющие сложные вычисления над различными данными:

·       архиваторы;

·       калькуляторы формул;

·       графопостроители;

·       системы распознавания;

·       модули шифрации-дешифрации…

В отличие от остальных групп задач здесь имеется направленность на выбор адекватных математических методов расчета и тестирование успешности работы сформированных алгоритмов для различных условий (данных).

 

К задачам автоматизации можно отнести приложения, автоматизирующие действия пользователя по работе с данными в таком ПО, как:

·       файловые менеджеры;

·       редакторы различного рода.

Например, редактор текстов с проверкой синтаксиса; редактор блок-схем с возможностью загрузки-формирования шаблона программы; редактор с возможностью автоматической коррекции фотографий для заданных условий и т.п.

В отличие от остальных групп задач здесь дается направленность в реализации автоматических алгоритмов обработки конкретных данных пользователя, а также интерфейсов для описания этих действий.

 

К задачам планирования можно отнести приложения, связанные с учетом данных, их анализом и выдачей рекомендаций:

·       задачи учета и распределения (домашняя бухгалтерия, магазин, транспорт);

·       секретари-планировщики для разного рода обслуживания (автомобиль, диета);

·       решение задач логистики (транспорт, перевозки грузов);

·       справочники-рекомендации (замена продуктов в рецепте, прием лекарств).

В отличие от остальных групп задач здесь дается направленность в формировании возможных вариантов развития событий по некоторым начальным данным.

 

К задачам моделирования можно отнести приложения, которые стремятся воспроизвести особенности развития событий по законам некоторого мира:

·       моделирование физических законов (симуляторы, модели разрушения объектов);

·       долговременное моделирование процессов развития группы объектов (развитие популяции живых организмов, прогнозирование погоды);

·       моделирования поведения (реализация «интеллекта» компьютерного противника).

В отличие от остальных групп задач здесь дается направленность в приближении текущего поведения объекта к некоторому миру (эксперимент, обучение или игра).
